# A check list for studing virus

## Gene

- Genomic map
- DNA or RNA gene
- Single or double stranded
- Positive or negative stranded
- Gene names and functions
- ORF
- Reference sequences
- Genotype or subtype sequences
- Recombinnation

## Structure

- Enveloped or non-enveloped
- Capsid
- Nucleocapside

## Life cycle

- Epitope
- Replication cycle
- Transcription mechanism

## Taxnonomy

- similar virus, related virus

## Single gene

- structure, PDB file
- conserved domian
- function prediction
- compare with similar virus same gene, the function
- Phylogenetic analysis for evolution history

## Lab methods

- culture virus
- permissive cell
- how to count the virus, which CPE (cypopathic effects) to look for.
- Sequencing method
- measure virus load
    - qPCR, RT-qPCR, digital PCR
- protein express
    - Assess protein expression via western blot, ELISA, or mass spectrometry

## Host virus interaction

- host receptor
- host immune response
- transcriptomic or proteomic change in infected cells
- disease and symptom

## Antiviral

- natural polymorphism of each mutations
- drug drug interaction

- transmission routes
- diagnostics

## Research data
